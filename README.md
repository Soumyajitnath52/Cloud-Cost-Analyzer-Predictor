# Cloud-Cost-Analyzer-Predictor
Cloud Cost Analyzer &amp; Predictor is a tool that monitors, analyzes, and forecasts cloud spending using historical usage data and machine learning. It identifies cost-saving opportunities, detects anomalies, and provides budget insights to optimize resource allocation and prevent overspending.
# Cloud Cost Predictor
Serverless ML API deployed on GCP Cloud Run

## Tech Stack
- GCP Cloud Run (Serverless)
- FastAPI + Scikit-learn
- Cost prediction accuracy: MAE $2.45
